# Statly — Discord Game Tracker

**Italian**

Statly è un bot Discord per tracciare rank e statistiche di gioco in **League of Legends**, **Valorant**, **Apex Legends** e **Rocket League**. Permette di registrare gli account dei giocatori, salvare gli snapshot giornalieri e inviare report programmati nei canali del server. I dati vengono salvati in SQLite (file `data/statly.db`).

**English**

Statly is a Discord bot that tracks ranks and game stats for **League of Legends**, **Valorant**, **Apex Legends**, and **Rocket League**. It lets players register their accounts, stores daily snapshots, and posts scheduled reports in server channels. Data is stored in SQLite (`data/statly.db`).

## Requirements

- Python 3.9+
- discord.py
- python-dotenv
- requests
- API keys (as needed): Riot API, Valorant (HenrikDev), Apex (Mozambique), Rocket League (RapidAPI)
